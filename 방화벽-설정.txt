Windows 방화벽에서 포트 열기
================================

1. Windows 검색창에서 "방화벽" 검색
2. "고급 보안이 포함된 Windows Defender 방화벽" 클릭
3. 왼쪽에서 "인바운드 규칙" 클릭
4. 오른쪽에서 "새 규칙..." 클릭

포트 3000 열기:
--------------
5. "포트" 선택 → 다음
6. "TCP" 선택
7. "특정 로컬 포트": 3000 입력 → 다음
8. "연결 허용" 선택 → 다음
9. 모든 프로필 체크 → 다음
10. 이름: "React 개발 서버" → 완료

포트 8081 열기:
--------------
11. 같은 방법으로 반복
12. 포트: 8081
13. 이름: "Spring Boot API 서버"


공유기 포트포워딩 설정 (외부 인터넷에서 접속하려면)
===========================================

1. 공유기 관리 페이지 접속 (보통 http://192.168.0.1 또는 http://192.168.1.1)
2. 로그인 (공유기 비밀번호)
3. "포트포워딩" 또는 "Virtual Server" 메뉴 찾기
4. 새 규칙 추가:
   - 서비스 포트(외부): 3000
   - 내부 IP 주소: (이 PC의 로컬 IP)
   - 내부 포트: 3000
   - 프로토콜: TCP
5. 한 번 더 추가:
   - 서비스 포트(외부): 8081
   - 내부 IP 주소: (이 PC의 로컬 IP)
   - 내부 포트: 8081
   - 프로토콜: TCP
6. 저장 및 재시작


보안 주의사항
============
- 로그인 기능이 있지만, 외부 공개 시 보안에 취약할 수 있습니다
- 가능하면 VPN을 통해 접속하는 것을 권장합니다
- HTTPS(SSL) 인증서가 없으면 데이터가 암호화되지 않습니다
